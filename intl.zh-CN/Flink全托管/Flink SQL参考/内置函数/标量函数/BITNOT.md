# BITNOT

本文为您介绍如何使用BITNOT函数进行按位取反。

**说明：** 本文仅适用于VVP 2.4.0且VVR 3.0.0及以上版本。

## 语法

```
INT BITNOT(INT number)     
```

**说明：** 输入和输出类型均为整型，且类型一致。

## 入参

|参数|数据类型|
|--|----|
|number|INT|

## 示例

-   测试数据

    |a\(INT\)|
    |--------|
    |7|

-   测试语句

    ```
    SELECT BITNOT(a) as var1
    FROM T1;             
    ```

-   测试结果

    |var1\(INT\)|
    |-----------|
    |-8|
